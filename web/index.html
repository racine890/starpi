Starpi Help & Documentation

üéØ What is Starpi?

Starpi is a Mock API Server that helps developers simulate backend responses during development. Perfect for frontend developers waiting for backend APIs or for testing with external services.

üöÄ Getting Started

  • Click "Add API" to create a new mock endpoint
  • Fill in the API details (name, method, endpoint)
  • Add JSON responses in the editor
  • Start the server and use your endpoints!

üîß Creating APIs

  • Endpoint: Must start with 'api' or 'download'
  • Path Variables: Use {*variable} syntax
  • Example: api/users/{*userId}/posts

üí° Managing Responses

  • Add multiple responses per API
  • Switch between responses in real-time
  • Use JSON editor with syntax highlighting
  • Responses are served in order of selection

üåê Using Your Mock APIs

Once the server is running, use tools like:

  • Postman or Insomnia
  • Your frontend application
  • cURL commands
  • Browser (for GET requests)

‚ö†Ô∏è Important Notes

  • Server runs on http://localhost:1901
  • All responses are public - don't use sensitive data
  • Path variables are flexible - any value will match
  • Headers and authentication are ignored for simplicity
